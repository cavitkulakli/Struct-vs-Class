
#  Struct vs Class

## Struct

- [x] Simpler
- [x] Faster
- [x] Deep copies
- [x] True immutability
- [x] No memory leaks
- [x] Threadsafe

## Classes

- [x] Has inheritance
- [x] Works with objective-C code

Apple advice about using [Struct or Classes](https://developer.apple.com/documentation/swift/choosing_between_structures_and_classes).
